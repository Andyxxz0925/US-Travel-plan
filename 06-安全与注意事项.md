## 1. 人身安全

### 🚨 总体安全原则

| 原则 | 详情 |
|------|------|
| 🌙 **天黑后谨慎** | 天黑后避免独自在偏僻街道行走，尽量 Uber 代步 |
| 📱 **财物贴身** | 手机 + 钱包贴身放，背包拉链朝身体侧 |
| 🪑 **不放桌上** | 在餐厅不要把手机放在桌子上（容易被抢走）|
| 👀 **保持警觉** | 走路时注意周围环境，不要一直低头看手机 |
| 🚗 **可疑远离** | 遇到可疑人员主动绕行，不要对视或起冲突 |
| 📸 **低调穿着** | 不要佩戴过于显眼的贵重首饰/名牌包 |

### 🚇 公共交通安全

| ⚠️ 注意 | 详情 |
|---------|------|
| 夜间乘车 | **晚上 10 点后** 尽量不坐地铁/公交，改用 Uber |
| 车厢选择 | 尽量坐在 **靠近司机或人多的车厢** |
| 等车位置 | 在站台 **灯光明亮、有摄像头** 的地方等车 |
| 行李看管 | 行李不要离开视线 |

### 🚗 Uber/Lyft 安全

```
上车前必做三件事：
┌─────────────────────────────────────────┐
│  ✅ 核对车牌号                           │
│  ✅ 核对司机名字和照片                    │
│  ✅ 核对车型和颜色                        │
└─────────────────────────────────────────┘
```

- ✅ 坐 **后排**
- ✅ 使用 APP 内 **"Share Trip"** 功能分享行程给朋友
- ❌ 绝对不要上 **没有匹配信息** 的车
- ❌ 如果司机问"你是不是 XXX？"，**不要直接说是**，反问"你接的是谁？"

## 3. 小费文化（完整指南）

> 💡 美国的小费不是可选的，是 **社会约定俗成的义务**。服务人员的基本工资很低，小费是收入的重要组成部分。

### 🍽️ 餐厅小费

| 场景 | 小费比例 | 计算方法 |
|------|:---:|------|
| **坐下来吃的餐厅** | **15~20%** | 按税前金额计算 |
| 服务非常好 | 20~25% | — |
| 服务一般 | 15% | — |
| 服务很差 | 10% | 最低也要给 |
| **快餐店 / 自取** | **0~5%** | 柜台可能有小费选项，可不给 |
| **外卖配送** | **15~20%** | — |
| **酒吧点酒** | **$1~2/杯** | 或总额的 15~20% |
| **咖啡店** | **$0~1** | 不强制，但友好 |

#### 📱 如何给小费？

```
方式 1：刷卡时 → 账单上会有 Tip 一栏 → 填写小费金额或百分比 → 写上 Total
方式 2：现金 → 直接放在桌上 → 走人
方式 3：iPad 屏幕 → 选择百分比（15%/18%/20%/自定义）→ 点选
```

#### 🧾 账单示例

```
╔════════════════════════════════════╗
║         RESTAURANT RECEIPT         ║
║                                    ║
║  Subtotal:     $45.00              ║
║  Tax (9%):      $4.05              ║
║  ──────────────────────            ║
║  Total:        $49.05              ║
║                                    ║
║  Tip: $_______ (建议填 $7~9)       ║
║                                    ║
║  Grand Total: $_______             ║
║                                    ║
║  Signature: _________________      ║
╚════════════════════════════════════╝
```

### 🏨 酒店小费

| 场景 | 小费金额 | 如何给 |
|------|:---:|------|
| **客房清洁** | $1~2/天 | 每天早上出门时放在枕头上或桌上，附纸条写 "Thank you" |
| **行李员** | $1~2/件 | 搬完行李后直接给 |
| **门童叫车** | $1~2 | 车到时给 |
| **Concierge 帮大忙** | $5~10 | 帮你订到难订的餐厅/票 |

### 🚗 交通小费

| 场景 | 小费金额 | 备注 |
|------|:---:|------|
| **Uber/Lyft** | $0~2 | APP 内可选，不给也行 |
| **出租车** | 15~20% | 或至少 $2 |
| **代客泊车 (Valet)** | $2~5 | 取车时给 |

### 💈 其他场景

| 场景 | 小费金额 |
|------|:---:|
| **理发** | 15~20% |
| **SPA/按摩** | 15~20% |
| **导游 (Tour Guide)** | $5~10/人 |

> ⚠️ **免小费场景**：快餐店（麦当劳等）、超市、自助结账、公共交通

## 4. 支付方式

### 💳 在美国怎么付钱？

| 优先级 | 方式 | 适用场景 | 备注 |
|:---:|------|------|------|
| ⭐1 | **Apple Pay** | 几乎所有商店、餐厅、地铁、公交 | 绑定 Visa 卡后最方便 |
| ⭐2 | **Visa/MC 实体卡** | Apple Pay 不支持的地方 | 插卡或刷卡 |
| 3 | **现金** | 小费、小店、紧急备用 | 带 $300~400 |

### 📱 Apple Pay 使用方法

```
Step 1 ─── 出发前在 iPhone "钱包" APP 中添加 Visa 借记卡
Step 2 ─── 到美国后，付款时双击侧边按钮 → Face ID 验证 → 靠近 POS 机
Step 3 ─── "叮！" 支付成功 ✅
```

### ⚠️ 支付注意事项

| 注意 | 详情 |
|------|------|
| **PIN码** | 借记卡可能需要 PIN 码（出发前在银行设置好）|
| **签名** | 信用卡小额交易无需签名，大额需签名 |
| **磁条/芯片** | 美国大部分已支持芯片卡（Chip），插卡即可 |
| **ATM取现** | 可以用 Visa 借记卡在 ATM 取美元现金（有手续费，约 $3/次）|

## 6. 文化差异与礼仪

### 🤝 社交礼仪

| 场景 | 美国习惯 |
|------|---------|
| **打招呼** | "Hi" / "How are you?" — 回答 "Good, thanks!" 即可，不用真的详说 |
| **排队** | 严格排队，不要插队 |
| **个人空间** | 和人保持 **一臂距离**，不要站太近 |
| **门** | 过门时帮后面的人扶一下门 |
| **谢谢** | 频繁说 "Thank you" / "Excuse me" / "Sorry" |
| **微笑** | 对视时微笑点头是礼貌 |

### 🚭 公共场所规则

| 规则 | 详情 |
|------|------|
| **禁烟** | 室内全面禁烟，餐厅/酒吧/商场都不能吸烟 |
| **饮酒** | 户外不能饮酒（Open Container 法），必须在餐厅/酒吧内 |
| **年龄** | 饮酒/买酒 **年满21岁**（可能被查 ID，带护照）|
| **噪音** | 公共交通上不要外放音乐/视频，不要大声讲电话 |
| **拍照** | 拍别人前最好问一句 "May I take a photo?" |

### 🚻 厕所

| 知识点 | 详情 |
|------|------|
| **叫法** | 说 "Restroom" 或 "Bathroom"（不说 Toilet/WC）|
| **哪里找** | 大商场、星巴克、麦当劳、图书馆 |
| **公共厕所** | 美国公共厕所较少，地铁站一般没有 |

## 7. 饮食注意

### 🥤 饮用水

| 知识点 | 详情 |
|------|------|
| **自来水** | 美国自来水可直接饮用 |
| **餐厅** | 坐下后服务员会免费提供冰水（Tap Water）|
| **瓶装水** | 超市买很便宜（$1~2/瓶）|

### 🍽️ 餐厅点餐流程

```
1. 进门 → 等 Host 带你入座（不要自己找位子坐）
2. 服务员来 → 先点饮料
3. 服务员再来 → 点主菜
4. 吃完 → 说 "Check, please" 要账单
5. 账单来 → 看清金额 → 写小费 → 签名
6. 离开
```

### 💡 省钱吃饭

| 方式 | 说明 | 预估花费 |
|------|------|:---:|
| **超市 + 便利店** | Trader Joe's、Whole Foods、7-Eleven | $5~10/餐 |
| **快餐** | McDonald's、Chipotle、In-N-Out | $8~12/餐 |
| **Food Court** | 商场美食广场 | $10~15/餐 |
| **正式餐厅** | 含小费 | $20~40/餐 |

## 9. 入境注意事项

### 🛂 入境流程

```
飞机降落 LAX
    ↓
跟指示牌走到 Customs & Border Protection
    ↓
排队等候（约 30~60 分钟）
    ↓
边检官员审查：
    ├── 出示护照 + 签证
    ├── 按指纹
    ├── 拍照
    └── 可能问几个问题（见下表）
    ↓
取托运行李
    ↓
过海关（一般走绿色通道 Nothing to Declare）
    ↓
出机场 ✈️→🏙️
```

### 📋 边检官可能问的问题

| 问题 | 建议回答 |
|------|---------|
| What's the purpose of your visit? | **Tourism** / Sightseeing |
| How long will you stay? | **About 11 days** |
| Where will you stay? | 说酒店名（最好打印确认单）|
| Do you have a return ticket? | **Yes**（打印或手机出示回程机票）|
| How much cash are you carrying? | 如实回答（<$10,000 无需申报）|

### ⚠️ 入境禁止携带

| ❌ 禁止 | 说明 |
|---------|------|
| 肉类、生鲜 | 包括牛肉干、鸡爪、火腿肠 |
| 水果、蔬菜 | 新鲜的不行 |
| 种子、植物 | 需申报 |
| 超过 $10,000 现金 | 需要申报（未申报违法）|
| 仿冒品 | 假名牌包等可能被没收 |
