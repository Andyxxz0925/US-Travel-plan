## 1. 银行卡办理

> 你的情况：在读硕士，无固定收入。必须在国内出发前办好，到美国无法办理（B签旅游签无SSN）。

### 你目前的进度

| 卡种 | 状态 | 说明 |
|------|:---:|------|
| **招行全币种信用卡** | ✅ 已申请 | 等待审批中（约7~14天出结果） |
| **中行 Visa 长城通借记卡** | ⏳ 明天去办 | 带身份证去中行大型支行，当天出卡 |

### 办理步骤（中行借记卡）

```
Step 1 — 带身份证去中国银行网点（选大型支行）
         告诉柜员"办长城跨境通 Visa 借记卡"
Step 2 — 当天出卡
Step 3 — 回家后往卡里充入美元
         方式A：去银行柜台购汇
         方式B：手机银行购汇转入（推荐，更方便）
```

### 招行信用卡跟进

```
如果审批通过 → 激活卡片 + 开通境外交易功能 + 设置密码
如果审批被拒 → 不用担心，中行借记卡完全够用
                也可以尝试建行龙卡畅享卡（学生可办）
```

### 注意事项

- 两张卡都带着，一张丢了还有一张备用
- 信用卡出发前开通境外交易功能
- 借记卡设置好 PIN 码（在美国 ATM 取现可能需要）
- 你的虚拟 MasterCard（用于海淘的那张）也可以带着作为第三备用

## 2. 电话卡

> 到美国后第一秒就需要网络（叫 Uber、查地图），所以必须提前在国内买好。

### 电话卡类型说明

| 类型 | 说明 | 适合你的需求？ |
|------|------|:---:|
| **实体 SIM 卡（含通话+短信+流量）** | 有真实美国手机号码，能打电话、收短信、上网 | ⭐ **首选推荐** |
| **纯流量 eSIM** | 只能上网，没有电话号码 | 不推荐（无法接收验证码） |
| **WiFi 蛋/移动热点** | 租赁设备，只能上网 | 不推荐（多带一个设备，没号码） |

### 关于美国真实电话号码

> **好消息：淘宝买的 T-Mobile 实体 SIM 卡自带一个真实的美国手机号码！**

| 问题 | 回答 |
|------|------|
| 我能获得美国电话号码吗？ | ✅ 可以！T-Mobile 实体 SIM 卡自带美国号码 |
| 这个号码能接收验证码吗？ | ✅ 可以，注册美国 APP 时可以收验证短信 |
| 能打电话叫外卖/联系酒店吗？ | ✅ 可以，和美国人的手机号一样用 |
| 能打回中国吗？ | 看套餐，大部分包含国际通话分钟数，或很便宜 |
| 号码能带回国继续用吗？ | ❌ 一般不行，30天有效期结束号码回收 |

### 推荐方案

| 方案 | 运营商 | 在哪买 | 价格 | 说明 |
|:---:|:---:|------|:---:|------|
| **首选** | **T-Mobile 30天不限流量卡** | 淘宝搜"美国电话卡 T-Mobile" | ￥80~150 | **含美国号码+通话+短信+不限流量**，落地插卡即用 |
| 备选 | **AT&T 15/30天卡** | 淘宝 | ￥100~180 | 信号覆盖更广，同样含号码+通话+流量 |

### 购买要点

- 淘宝搜索关键词：**"美国实体SIM卡 T-Mobile 30天"**
- ⚠️ 确认包含以下全部：
  - [x] 4G/5G 不限量流量
  - [x] 美国本地通话（无限分钟）
  - [x] 美国本地短信（无限条）
  - [x] 真实美国手机号码
- 不要买"纯流量卡"或"eSIM"，那种没有电话号码
- 提前3-5天下单，邮寄到手
- 在飞机上关机前换好 SIM 卡，落地开机即用

### 你的一加15手机注意事项

| 注意 | 说明 |
|------|------|
| SIM卡制式 | T-Mobile用Nano SIM，一加15支持，没问题 |
| 网络频段 | 一加15支持美国主要4G/5G频段，可以正常使用 |
| 换卡方法 | 关机 → 用取卡针弹出SIM卡槽 → 换上T-Mobile卡 → 开机 |
| 原卡保管 | 把中国SIM卡放在安全的地方（钱包里），回国换回来 |

#### 🔥 关于 Google Pay vs Apple Pay 的详细说明

> 你的一加15是安卓+谷歌架构，无法使用 Apple Pay，但 **Google Pay 在美国完全够用！**

| 问题 | 回答 |
|------|------|
| Google Pay 在美国好用吗？ | ✅ **非常好用！** 和 Apple Pay 使用同样的 NFC 感应支付终端 |
| 支持的地方一样多吗？ | ✅ 是的！所有支持 Apple Pay 的 POS 机都支持 Google Pay（因为都是 NFC 协议） |
| 为什么感觉 Apple Pay 更常见？ | 美国iPhone用户多，所以商家标牌上常写Apple Pay，但NFC终端对Google Pay同样有效 |
| 我能绑定虚拟 MasterCard 吗？ | ✅ 可以！Google Pay 支持绑定 MasterCard |
| 公共交通能用吗？ | ⚠️ 看城市（详见下表） |
| 线上支付呢？ | ✅ Google Pay 支持很多线上商家的快捷支付 |

#### 各城市交通卡对 Google Pay 的支持

| 城市 | 交通卡 | Google Pay | Apple Pay | 建议 |
|:---:|:---:|:---:|:---:|------|
| 洛杉矶 | TAP 卡 | ✅ 支持虚拟TAP卡 | ✅ 支持 | 手机直接刷 |
| 旧金山 | Clipper 卡 | ✅ 支持虚拟Clipper卡 | ✅ 支持 | 手机直接刷 |
| 圣地亚哥 | PRONTO 卡 | ⚠️ 可NFC支付 | ⚠️ 可NFC支付 | 建议买实体卡保险 |
| 西雅图 | ORCA 卡 | ⚠️ 可NFC支付 | ⚠️ 可NFC支付 | 建议买实体卡保险 |
| 拉斯维加斯 | 无统一卡 | — | — | 公交买票/Uber |

> 🎯 **结论：Google Pay 和 Apple Pay 在美国的适用范围几乎完全相同，放心用！**

#### 你的 iPhone XS 能用 Apple Pay 吗？

| 问题 | 回答 |
|------|------|
| iPhone XS 支持 Apple Pay 吗？ | ✅ **完全支持！** iPhone XS 有 NFC + Face ID |
| 能绑定 Visa/MasterCard 吗？ | ✅ 可以 |
| 能用于交通刷卡吗？ | ✅ 可以（LA TAP、SF Clipper 等） |
| 建议怎么用？ | 带着作为**备用支付设备** — 如果一加15没电或遇到问题，掏出 iPhone XS 用 Apple Pay |

#### 支付方案总结

```
主力支付：一加15 + Google Pay（绑定 Visa借记卡 + 虚拟MasterCard）
备用支付：iPhone XS + Apple Pay（绑定 Visa借记卡）
第三备用：实体 Visa/MasterCard 插卡刷
最后手段：现金
```

#### Google Pay 设置步骤

```
Step 1 — 在一加15上打开 Google Pay（Google Wallet）APP
Step 2 — 点击"添加支付方式" → 添加 Visa 借记卡 + 虚拟 MasterCard
Step 3 — 银行验证（可能需要短信验证码）
Step 4 — 到美国后，付款时解锁手机 → 靠近 POS 机（有 NFC/非接触标志的）→ 支付成功

注意：部分手机需要在设置中开启NFC功能
一加15：设置 → 连接与共享 → NFC → 开启
```

> 详细的航班时间和时差解释请查看 → [航班与时差说明](航班与时差说明.md)
> 转机相关手续请查看 → [转机指南](转机指南.md)

### 航线方案

```
去程：上海浦东 PVG → 洛杉矶 LAX（4/15 约20:00出发）
回程：西雅图 SEA → 上海浦东 PVG（4/29 约15:00出发）
```

### 你的去程

| 项目 | 详情 |
|------|------|
| **出发** | 上海浦东 4月15日 ~20:00（北京时间） |
| **到达** | 洛杉矶 LAX 4月15日 ~17:00（美西时间，同一天！）|
| **飞行时间** | 约12小时 |
| **为什么同一天？** | 时差15小时 > 飞行12小时，所以日期不变 |

### 你的回程

| 项目 | 详情 |
|------|------|
| **出发** | 西雅图 SEA 4月29日 ~15:00（美西时间） |
| **到达** | 上海浦东 4月30日 ~18:00-20:00（北京时间）或5月1日（转机） |
| **飞行时间** | 直飞约11h / 转机约15-20h |

### 价格参考

| 航段 | 方式 | 参考价格 | 平台 |
|------|------|:---:|------|
| 上海 → 洛杉矶 | 直飞 / 转机 | ¥3,000 ~ 5,000 | 携程 / Google Flights / Skyscanner |
| 西雅图 → 上海 | 直飞海航 / 转机 | ¥3,500 ~ 5,500 | 同上 |
| **往返合计** | 开口票 | **¥6,500 ~ 10,000** | — |

### 省钱技巧

- 上海出发航班最多、价格最优，建议上海出发
- 用 Google Flights 设价格提醒，降价自动通知
- 转机（东京/首尔中转）通常比直飞便宜 ¥1,000~2,000（转机注意事项见[转机指南](转机指南.md)）
- 周中出发/返回通常比周末便宜
- 提前 30~45天购买是价格甜点区

### 携程上时间怎么看？

> **所有机票显示的时间都是当地时间！**

```
去程显示：
  上海 PVG ~20:00  ← 北京时间
  洛杉矶 LAX ~17:00  ← 美西时间（同一天，没标"+1"）
```

## 5. EVUS登记确认

> 持有10年美国B签证的中国护照持有人，必须进行 EVUS 登记。

- 登录 [evus.gov](https://www.evus.gov) 确认登记状态为"已登记"
- 如已过期需重新登记（免费，10分钟在线完成）
- EVUS 有效期为2年，或护照/签证到期时失效

## 7. 现金准备

> 美国 95% 的地方可以刷卡 / Apple Pay，但部分场景需要现金。

### 换汇建议

| 面额 | 数量 | 用途 |
|:---:|:---:|------|
| $1 | × 20张 | 小费（行李员、酒店清洁、停车代客等）|
| $5 | × 5张 | 小额现金消费 |
| $20 | × 10张 | 备用（小店、紧急情况）|
| **合计** | **$270~400** | |

### 换汇方式

1. 去银行柜台直接购汇（说"买美元现钞"）
2. 或手机银行购汇 → 去柜台取现钞
3. 要求小面额（大面额 $100 的有些小店不收）
