## 3. The Strip（拉斯维加斯大道）

> The Strip 就是 Las Vegas Boulevard，全长约 4.2 英里（6.8公里），是拉斯维加斯的灵魂。

### Strip 上的标志性酒店/赌场（从南到北）

| 酒店/赌场 | 主题/特色 | 必看项目 |
|----------|---------|---------|
| **Luxor** | 埃及金字塔造型 | 外观拍照 |
| **New York-New York** | 纽约天际线缩小版 + 过山车 | 外观拍照 |
| **MGM Grand** | 超大型赌场 | 大堂逛逛 |
| **Bellagio** | 意大利风情 | ⭐ **喷泉秀**（免费，每15-30分钟一次）|
| **Caesars Palace** | 古罗马主题 | The Forum Shops 购物街（室内人造天空）|
| **The Venetian** | 威尼斯主题 | ⭐ 室内运河 + 贡多拉船 + 仿圣马可广场 |
| **The Mirage** | 热带主题 | 火山秀（如开放） |
| **Wynn / Encore** | 顶级奢华 | 花园 + 艺术品 |

### 免费值得看的

| 项目 | 地点 | 说明 |
|------|------|------|
| **Bellagio 喷泉秀** | Bellagio 门前 | 白天每30分钟，晚上每15分钟，配音乐，免费 |
| **Bellagio 温室花园** | Bellagio 内 | 季节性花卉展览，免费，拍照绝佳 |
| **Caesars Palace Forum Shops** | Caesars 内 | 购物+仿古罗马街道+人造天空 |
| **Venetian 大运河** | Venetian 内 | 免费参观，坐船另付费（~$30） |
| **Fremont Street Experience** | Downtown | 超大天幕 LED 灯光秀，免费 |
| **Welcome to Las Vegas Sign** | Strip 南端 | 经典拍照打卡点 |

## 4. 必去景点与体验

### High Roller（世界最大摩天轮）

| 项目 | 详情 |
|------|------|
| **位置** | LINQ Promenade（Strip中段） |
| **高度** | 167米（55层楼高） |
| **一圈时间** | 30分钟 |
| **票价** | 白天 ~$25, 夜晚 ~$35（夜景更值） |
| **推荐** | ⭐ 夜晚坐，俯瞰整个 Strip 灯火通明 |

### Fremont Street Experience（弗里蒙特街）

| 项目 | 详情 |
|------|------|
| **位置** | Downtown（离 Strip 北端约15分钟车程）|
| **特色** | 巨型 LED 天幕灯光秀 + 老式霓虹灯 + 街头表演 |
| **费用** | 免费 |
| **推荐时间** | 天黑后去，灯光效果最好 |
| **交通** | 从 Strip 坐 Deuce 公交或 Uber ~$10 |

### 大峡谷一日游（可选）

| 目的地 | 距离 | 方式 | 费用 | 说明 |
|------|:---:|------|:---:|------|
| **Grand Canyon West（西缘）** | 2.5h车程 | 一日游团 | $100~200 | 有天空步道（Skywalk），最近 |
| **Grand Canyon South（南缘）** | 4.5h车程 | 一日游团 | $100~150 | 景色最经典，但路远 |
| **Red Rock Canyon** | 30min车程 | Uber $25 | $15入园 | 红色岩石奇观，半天够 |

> 建议：如果时间有限（你有2天），Red Rock Canyon 半日游更实际。大峡谷要一整天。

## 5. PRO Gun Club 射击体验 🔫

> PRO Gun Club 是拉斯维加斯近郊的专业射击场，提供面向游客的射击体验。

### 基本信息

| 项目 | 详情 |
|------|------|
| **名称** | PRO Gun Club（PRO Gun Vegas） |
| **地址** | 12801 US 95 South, Boulder City, NV 89005 |
| **距Strip** | 开车约30分钟（Uber ~$25） |
| **电话** | (702) 293-2108 |
| **营业时间** | 周一至周日 9:00 AM ~ 5:00 PM（建议出发前致电确认：(702) 293-2108）|
| **预约** | 不需要预约，walk-in 即可（建议工作日去人少） |
| **年龄要求** | 必须年满21岁 |
| **证件** | 需携带护照（外国游客用护照作为身份证件） |

### 射击套餐参考

| 套餐 | 价格 | 包含内容 |
|------|:---:|------|
| **手枪入门体验** | $50~80 | 1-2把手枪 + 弹药 + 靶纸 + 安全培训 |
| **步枪+手枪组合** | $80~120 | 手枪+步枪（如AR-15）+ 弹药 + 培训 |
| **VIP全套体验** | $120~150+ | 多种武器体验 + 更多弹药 + VIP待遇 |

> 💡 以上价格为参考，实际以现场为准。额外弹药另购。

### 射击流程

```
Step 1 — 到达后在前台签到 + 出示护照
Step 2 — 选择射击套餐并付款
Step 3 — 安全培训（约15分钟，英语，教你如何安全操作枪械）
Step 4 — 佩戴护目镜和耳罩
Step 5 — 在教练全程指导下射击（教练会帮你装弹、调整姿势）
Step 6 — 射击结束 → 可以拍照留念（在安全区域）
Step 7 — 整个体验约 1~1.5 小时
```

### 射击场常用英语

```
"I'd like to try the pistol experience." — 我想试试手枪体验。
"Is there a package for beginners?" — 有新手套餐吗？
"I've never shot a gun before." — 我以前没开过枪。
"Can I take photos after?" — 我可以之后拍照吗？
"How much for extra ammo?" — 额外弹药多少钱？
```

### 安全须知

| 注意事项 | 详情 |
|---------|------|
| **听从教练指挥** | 全程按教练指示操作，不要自行行动 |
| **枪口永远朝向靶区** | 不管有没有子弹，枪口永远不能对人 |
| **不要触碰扳机** | 不准备射击时手指不能放在扳机上 |
| **佩戴防护装备** | 护目镜和耳罩必须全程佩戴 |
| **支付方式** | 接受信用卡/借记卡/Google Pay，也收现金 |
| **穿着** | 穿封闭式鞋子（不要穿拖鞋/凉鞋），避免低领上衣 |

## 6. 赌场参观指南

> 你说想"赌场外围参观"——完全可以！赌场大厅都是免费开放的。

### 参观规则

| 规则 | 说明 |
|------|------|
| **可以自由进出** | 赌场大厅对所有人开放，不需要消费 |
| **最低年龄** | 必须满21岁才能进入赌场区域（可能查ID） |
| **拍照** | 大厅、装饰可以拍，**不要对着赌桌和玩家拍** |
| **穿着** | 没有严格着装要求（T恤牛仔裤OK），但不能太邋遢 |
| **饮料** | 在赌桌前赌博的人可以获得免费饮料（你不赌就不适用） |

### 推荐参观路线（不赌钱版）

```
从 Strip 南端往北走：

1. 在 "Welcome to Las Vegas" 牌子前拍照
2. 路过 Luxor 金字塔 → 外观拍照
3. New York-New York → 外观拍照（缩小版自由女神像）
4. 进入 Bellagio → 看温室花园 → 出来看喷泉秀
5. 进入 Caesars Palace → 逛 Forum Shops → 感受古罗马主题
6. 进入 The Venetian → 看室内运河和仿威尼斯广场
7. 在 LINQ 坐 High Roller 摩天轮看夜景

全程步行约 3-4 小时（Strip 很长，注意防晒和补水）
```

### 如果想小试一下

| 游戏 | 最低赌注 | 建议 |
|------|:---:|------|
| 老虎机（Slot Machine） | $0.01起 | 投个$5-10感受下，纯娱乐 |
| 轮盘（Roulette） | $5~10 | 规则简单，红或黑、奇或偶 |
| 21点（Blackjack） | $10~25 | 最经典的桌游，规则需要提前了解 |

> ⚠️ 赌博有风险，建议设定预算上限（比如 $20-50），花完就走，纯当娱乐。

## 8. 市内交通

| 方式 | 覆盖 | 费用 | 说明 |
|------|------|:---:|------|
| **步行** | Strip 上 | 免费 | Strip 很长（6.8km），做好走路准备 |
| **Deuce 公交** | Strip 全线 + Downtown | $6/2小时, $8/24小时 | 沿 Strip 运行，方便 |
| **Monorail 单轨** | Strip 东侧部分酒店 | $5/次, $13/24小时 | 覆盖有限 |
| **Uber/Lyft** | 全城 | $8~20 | Strip 内短途也可以用 |
| **酒店间免费电车** | 部分酒店之间 | 免费 | Bellagio ↔ Vdara, Mandalay Bay ↔ Luxor ↔ Excalibur |

### 交通建议

- Strip 上白天晒，建议 Uber 短途或利用酒店内走（酒店之间很多有室内通道）
- 去 Fremont Street：Deuce 公交或 Uber（~$10-15）
- 去机场：Uber（~$10-15，非常近）
- 去 Red Rock Canyon：Uber 或租车

## 9. 住宿建议

> 拉斯维加斯酒店非周末（周日-周四）超级便宜！

| 酒店 | 位置 | 平日价格 | 特点 |
|------|------|:---:|------|
| **The LINQ** | Strip 中段 | $40~80 | 性价比高，位置好，旁边就是 High Roller |
| **Flamingo** | Strip 中段 | $30~70 | 经典老牌，泳池不错 |
| **Excalibur** | Strip 南段 | $25~60 | 最便宜选择之一 |
| **Park MGM** | Strip 南段 | $50~100 | 比较新，环境好 |
| **Venetian**（推荐体验） | Strip 中段 | $100~200 | 全套房，豪华体验 |

> 💡 4月平日（周一到周四）是淡季，五星级酒店可能只要 $50-80！周末会贵2-3倍。

### 住宿额外费用

| 费用 | 金额 | 说明 |
|------|:---:|------|
| **Resort Fee（度假费）** | $30~50/晚 | 几乎所有 Strip 酒店都收，订房时注意 |
| 停车 | $15~20/天 | 自驾才需要 |

> 注意：LV 酒店报价通常不含 Resort Fee 和税，实际价格要加 $40-60/晚。

## 11. 注意事项

| 注意事项 | 详情 |
|---------|------|
| **防晒+补水** | 沙漠气候极干燥，4月白天可达30°C，随身带水 |
| **赌场21岁** | 进赌场区域需满21岁，带好护照备查 |
| **Strip很长** | 看着不远实际要走很久，穿舒适的鞋 |
| **免费演出时间** | Bellagio 喷泉：周一到周五下午3点开始，周末中午开始 |
| **小费** | 和其他城市一样 |
| **安全** | Strip 上很安全；Strip 以外的街区天黑后注意安全 |
| **买酒21岁** | 会查 ID，随身带护照 |
| **拍照** | 酒店大厅随便拍，赌桌区域不要拍 |
