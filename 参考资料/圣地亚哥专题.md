## 1. 为什么值得去？

| 亮点 | 说明 |
|------|------|
| **完美气候**| 4月日均 18~24°C，几乎天天晴天，全美最佳气候城市之一 |
| **绝美海滩**| La Jolla Cove 悬崖海岸 + Coronado Beach 全美最美沙滩 |
| **野生动物**| La Jolla 的野生海豹群，免费近距离观看 |
| **文化底蕴**| Balboa Park 西班牙殖民建筑 + 17个博物馆 |
| **军事历史**| USS Midway 中途岛航母——震撼的退役航母博物馆 |
| **正宗墨西哥美食**| 靠近墨西哥边境，Taco 比 LA 更正宗 |
| **安全**| 比洛杉矶和旧金山更安全，Solo Trip 更放心 |
| **性价比**| 住宿餐饮比 LA 便宜 20~30% |

## 2. 必去景点详解

### La Jolla Cove（拉霍亚海湾）

```
必去指数：MAX
```

| 项目 | 详情 |
|------|------|
| **亮点**| 悬崖海岸线 + 野生海豹群 + 海景步道 + 海蚀洞 |
| **门票**| 免费 |
| **需要时间**| 2~3 小时 |
| **交通**| Uber ~$15-20（无直达公交）|
| **建议**| 上午去光线最好，带防晒霜 |

#### 路线建议

```
La Jolla Cove 悬崖观景台 → 沿海步道向南走
→ Children's Pool（海豹聚集地 ）
→ La Jolla Shores（宽阔沙滩 ）
→ 在 La Jolla Village 吃海鲜午餐 
```

#### 拍照tips

- 悬崖顶端可俯拍碧蓝海水 + 海豹群
- 日出时分光线最美（但上午也很好）
- 不要试图触摸海豹！保持 **至少3米**距离

### Balboa Park（巴尔博亚公园）

```
必去指数：MAX
```

| 项目 | 详情 |
|------|------|
| **亮点**| 西班牙殖民建筑群 + 17个博物馆 + 花园 + 免费表演 |

| **需要时间**| 半天（3~4 小时）|

|------|------|:---:|
| **El Prado 主步道**| 西班牙殖民风格建筑长廊，拍照绝佳 | |
| **Botanical Building 植物馆**| 全美最大的户外木结构建筑之一 | |
| **Lily Pond 荷花池**| 网红拍照点，倒影超美 | |
| **Japanese Friendship Garden**| 日式花园，宁静 | $14 |
| **San Diego Museum of Art**| 欧洲和亚洲艺术收藏 | $20 |
| **Air & Space Museum**| 航空航天博物馆 | $24 |

### USS Midway 中途岛航母博物馆

```
必去指数：MAX
```

| 项目 | 详情 |
|------|------|
| **亮点**| 退役航母实地参观 + 战斗机 + 舰桥 + 飞行模拟器 |
| **门票**| **$33**（网上买可能有折扣）|
| **需要时间**| 2~3 小时 |
| **交通**| 从 Gaslamp Quarter 步行 10 分钟 |
| **语音导览**| **有中文版！**免费随票附赠 |

#### 参观路线建议

```
Flight Deck（飞行甲板）→ 各型战斗机、直升机
→ Hangar Deck（机库甲板）→ 展览和模拟器
→ Bridge（舰桥指挥室）→ 感受舰长视角
→ 甲板上拍 San Diego 天际线 
```

## 3. 从洛杉矶怎么去？

### 首选：Amtrak Pacific Surfliner 火车

| 项目 | 详情 |
|------|------|
| **出发站**| LA Union Station |
| **到达站**| San Diego Santa Fe Depot（市中心！）|
| **车程**| 约 2.5 小时 |
| **班次**| 每天 13 班 |
| **票价**| $32~40（普通座）|
| **购票**| Amtrak App 或 amtrak.com |

> **全程沿太平洋海岸行驶，风景壮丽！坐左侧靠窗座！**

## 4. 市内交通

| 交通方式 | 覆盖范围 | 票价 |
|---------|---------|:---:|
| **MTS Trolley**| 市中心 ↔ Old Town ↔ 边境 | $2.50 |
| **MTS Bus**| 全市 | $2.50 |
| **Uber/Lyft**| 全市 | 按距离 |
| **步行**| Gaslamp Quarter + 港口区 | |

> 交通卡：**PRONTO 卡**（Trolley 站自助售卡机购买）

### 需要 Uber 的景点

| 目的地 | 从 Gaslamp Quarter 出发 | Uber 费用 |
|------|------|:---:|
| La Jolla Cove | 无便捷公交 | $15~20 |
| Balboa Park | 可坐公交，Uber更快 | ~$10 |
| Coronado Beach | Bus 901 可到 | $10~15 |

## 5. 美食推荐

> 圣地亚哥靠近墨西哥边境，墨西哥美食是全美最正宗的！

| 类型 | 推荐 | 在哪吃 | 价格 |
|------|------|------|:---:|
| **Fish Taco**| 炸鱼 Taco（圣地亚哥特色！）| Old Town / Gaslamp | $3~5/个 |
| **Burrito**| 超大墨西哥卷饼 | Old Town | $8~12 |
| **Seafood**| 新鲜海鲜 | La Jolla / Seaport Village | $15~25 |
| **Churros**| 西班牙油条（蘸巧克力酱）| Old Town | $4~6 |
| **Craft Beer**| 精酿啤酒（圣地亚哥是精酿之都！）| Gaslamp Quarter | $6~10 |
