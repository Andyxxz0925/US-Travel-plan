| Amtrak 火车 | ¥500 | $70 | LA → SD + SD → LA（往返） |

### 总计

**总预算：¥21,500 ~ ¥34,000**

| 预算等级 | 金额 | 说明 |
|---------|:---:|------|
| 节省模式 | ~¥21,500 | 转机国际机票+经济住宿+控制餐饮+基础球票 |
| 舒适模式 | ~¥27,000 | 直飞+中档住宿+正常餐饮+好座位球票 |
| 丰俭由人 | ~¥34,000 | 直飞+品质住宿+自由消费 |

> 💡 控制预算关键：国际机票选转机、住宿选性价比高的、LV 选平日（你是周二三住，最便宜！）

## 4. 省钱技巧

### 机票省钱

| 技巧 | 省多少 |
|------|:---:|
| 选转机（东京/首尔）而非直飞 | 省 ¥1,000~2,000 |
| 周中出发/返回 | 省 ¥500~1,000 |
| 提前 30~45 天购买 | 最优价格区间 |
| 用 Google Flights 设价格提醒 | — |
| 境内选 Southwest（免费行李）| 省 $30~60 |

### 住宿省钱

| 技巧 | 省多少 |
|------|:---:|
| 选 Airbnb 而非酒店（各城市通用）| 每晚省 $20~40 |
| 有厨房的 Airbnb 可自己做饭 | 每天省 $10~15 |
| 住 Koreatown（LA）性价比最高 | 比 Hollywood 便宜 $20/晚 |

### 餐饮省钱

| 技巧 | 省多少 |
|------|:---:|
| 早餐超市自解决 | 每天省 $5~8 |
| 午餐选快餐/Food Court | — |
| In-N-Out（加州特有快餐）性价比极高 | $6~8 吃饱 |
| Chipotle 大份量墨西哥卷 | $10 管饱 |
| 自己带水杯（美国自来水可直饮）| 省水钱 |

### 交通省钱

| 技巧 | 省多少 |
|------|:---:|
| 优先公共交通，Uber 补充 | 每天省 $10~20 |
| Uber 和 Lyft 比价 | 有时差价 $5~10 |
| 避开高峰 Surge Pricing | 省 30~50% |
| 旧金山多走路（城市紧凑）| — |
