## 1. 转机的核心概念

### 什么是"联程票"和"非联程票"

| 类型 | 说明 | 行李处理 | 误机保障 |
|------|------|---------|---------|
| **联程票**（推荐） | 一张票包含所有航段，在同一个订单里 | 行李直挂目的地，转机时不用取行李 | 如果前段延误导致误后段，航司负责改签 |
| **非联程票** | 分开买的两张独立机票 | 需要在转机城市取行李、重新值机、重新托运 | 前段延误不管后段，风险自担 |

> ⚠️ **强烈建议买联程票！** 买的时候确认是一个订单出的，行李可以直挂。

### 转机 vs 经停

| 类型 | 说明 |
|------|------|
| **转机（Transfer）** | 需要下飞机，换另一架飞机，可能换航站楼 |
| **经停（Stopover/Layover）** | 飞机中途降落加油/上下客，你可能不用下飞机或短暂下飞机 |

## 2. 香港转机

### 基本信息

| 项目 | 详情 |
|------|------|
| **机场** | 香港国际机场 HKG |
| **是否需要港澳通行证？** | 看情况（详见下方） |
| **转机区域** | 有国际转机通道，可不入境香港 |

### 两种情况

#### 情况一：不出机场（纯国际转机）

| 条件 | 说明 |
|------|------|
| **是否需要港澳通行证** | ❌ 不需要 |
| **需要什么** | 中国护照 + 美国签证 + 联程机票（同一订单）+ 前后航段登机牌 |
| **流程** | 下飞机 → 跟"Transfer"指示牌 → 过安检 → 到下一个登机口 |
| **行李** | 联程票行李直挂，不用管 |
| **注意** | 必须在同一天内完成转机，不能过夜停留在转机区 |

#### 情况二：需要入境香港（过夜转机/想出去逛）

| 条件 | 说明 |
|------|------|
| **是否需要港澳通行证** | ⚠️ 需要，或者使用"过境免签" |
| **过境免签条件** | 持有效中国护照 + 前往第三国的机票（你的美国机票），可免签在港停留 **7天** |
| **实际操作** | 下飞机 → 走入境通道 → 出示护照+美国机票 → 入境香港 |
| **注意** | 回程不适用此规则（从美国回来经停香港不算"前往第三国"），需要港澳通行证 |

### 香港转机建议

- ✅ 联程票 + 转机时间2-4小时 → 不出机场，纯走转机通道，最简单
- ⚠️ 过夜转机 → 确认是否能留在转机区域过夜，或需要入境
- 🔧 香港机场转机设施很好，有餐厅、休息区、免税店

## 3. 首尔仁川转机

### 基本信息

| 项目 | 详情 |
|------|------|
| **机场** | 首尔仁川国际机场 ICN |
| **最常见航司** | 大韩航空 KE、韩亚航空 OZ |
| **转机设施** | 非常好，免税店、免费转机休息室、免费文化体验 |

### 是否需要韩国签证？

#### 情况一：不出机场（纯转机）

| 条件 | 说明 |
|------|------|
| **是否需要签证** | ❌ 不需要 |
| **流程** | 下飞机 → "Transfer"通道 → 过安检 → 到下一个登机口 |
| **适用条件** | 联程票，转机时间在同一天内 |

#### 情况二：想出机场逛逛（入境韩国）

| 条件 | 说明 |
|------|------|
| **是否需要签证** | ⚠️ 取决于政策，目前中国护照持有人持有效美国签证可以申请韩国过境免签（TWOV），通常允许停留30天 |
| **注意** | 此政策可能变化，出发前务必在韩国驻华大使馆官网确认最新政策 |
| **流程** | 下飞机 → 走入境通道 → 出示护照+美签+后续机票 → 入境韩国 |

### 首尔转机建议

- ✅ 联程票 + 2-4小时转机时间 → 不出机场最简单
- 仁川机场转机区有免费淋浴、免费文化体验（韩服试穿等），足够消磨时间
- 如果转机超过6小时且想出去 → 提前查确认免签政策

## 4. 东京转机

### 基本信息

| 项目 | 详情 |
|------|------|
| **机场** | 成田 NRT（国际航班为主）或 羽田 HND |
| **最常见航司** | ANA（全日空）、JAL（日本航空） |
| **注意** | 如果两段航班分属成田和羽田，需要换机场（需入境）！ |

### 是否需要日本签证？

#### 情况一：不出机场（同一机场内转机）

| 条件 | 说明 |
|------|------|
| **是否需要签证** | ❌ 不需要 |
| **条件** | 两段航班在**同一个机场**，且有联程票 |
| **流程** | 下飞机 → 国际转机通道 → 过安检 → 下一个登机口 |
| **行李** | 联程票行李直挂 |

#### 情况二：需要入境（换机场或过夜）

| 条件 | 说明 |
|------|------|
| **是否需要签证** | ⚠️ 中国护照需要日本签证才能入境 |
| **Shore Pass（临时入境许可）** | 日本有时发放 Shore Pass 给转机乘客临时入境，但**不保证**，由入境官员决定 |
| **建议** | 如果需要换机场（成田↔羽田）或过夜转机，**强烈建议提前办日本签证**或换其他转机方案 |

### 东京转机建议

- ✅ 选择同一机场转机的联程票（全日空/日航通常安排在同一机场）
- ⚠️ 不要买需要换机场的票（成田转羽田），除非你有日本签证
- 成田/羽田转机区都有不错的设施和免税店

## 5. 美国境内转机

> 这是很多人担心的部分，其实比想象中简单！

### 核心规则

> **入境美国时，你在第一个到达的美国机场办理入境手续。之后的美国国内航段，和普通国内乘客一样，不再过海关。**

### 场景一：直飞到 LAX（你的情况）

```
你买的是直飞上海 → 洛杉矶的票

流程：
  飞机降落 LAX
      ↓
  下飞机，跟指示牌走到 Immigration（入境审查）
      ↓
  排队等候（30-60分钟）
      ↓
  边检官员：出示护照+签证 → 按指纹 → 拍照 → 问几个问题
      ↓
  取托运行李
      ↓
  过海关（通常走 Nothing to Declare 绿色通道）
      ↓
  出机场 → 你在美国了！✅

后续在美国坐飞机（如 SD→SF, SF→SEA）：
  → 和美国人坐飞机一样
  → 出示护照（作为身份证件）
  → 过 TSA 安检
  → 登机
  → 不需要再过海关
```

### 场景二：转机到达美国（如 上海 → 首尔 → 洛杉矶）

```
上海 → 首尔：普通国际转机（见上文首尔部分）
首尔 → 洛杉矶：到达LAX后，正常入境美国（和直飞一样）

注意：联程票的话，行李可能在上海就直挂到了洛杉矶
      到了LAX取行李 → 过海关 → 出机场
```

### 场景三：在美国境内转机（如 纽约转机到洛杉矶）

```
如果你买的票是 上海 → 纽约 → 洛杉矶

在纽约（第一个到达的美国城市）：
  → 下飞机 → 过入境审查（海关、移民局）
  → 取托运行李
  → 过海关
  → 重新托运行李（如果联程票，有专门的转机行李柜台）
  → 转国内航班区域
  → 过 TSA 安检
  → 找到下一段登机口
  → 登机飞洛杉矶

⚠️ 在美国，即使是联程票，入境时也要取行李过海关，然后重新托运！
   这和在亚洲转机不同，请预留至少 2-3 小时的转机时间。
```

### 美国入境的特殊规则

| 规则 | 说明 |
|------|------|
| **必须取行李** | 无论联程票还是非联程票，在美国第一个入境城市必须取行李过海关 |
| **重新托运** | 过完海关后，有专门的"Connecting Flights"行李柜台，把行李放上去 |
| **重新安检** | 需要再过一次 TSA 安检 |
| **预留时间** | 美国境内转机至少预留 **2.5~3小时** |

## 6. 转机对比总结

| 转机城市 | 需要签证吗？ | 转机便利度 | 机场设施 | 推荐指数 |
|---------|:---:|:---:|:---:|:---:|
| **首尔仁川** | 不出机场不需要 | ⭐⭐⭐⭐⭐ | 极佳 | ⭐⭐⭐⭐⭐ |
| **东京成田** | 同机场转不需要 | ⭐⭐⭐⭐ | 很好 | ⭐⭐⭐⭐ |
| **香港** | 不出机场不需要 | ⭐⭐⭐⭐ | 极佳 | ⭐⭐⭐⭐ |
| **美国城市转机** | 已有美签 | ⭐⭐⭐ | 一般 | ⭐⭐⭐ |

### 推荐排序

1. **首尔仁川** — 航司选择多、免签政策好、机场设施顶级
2. **东京成田** — 航班多、但注意别买换机场的票
3. **香港** — 方便、设施好、但回程转机可能需要通行证
4. **美国境内转机** — 流程最繁琐（取行李+过海关+重新安检），不推荐新手

## 7. 转机注意事项与建议

### 通用建议

| 建议 | 说明 |
|------|------|
| **买联程票** | 一个订单出票，行李直挂，误机有保障 |
| **转机时间** | 国际转国际：至少 **2小时**；美国入境转机：至少 **3小时** |
| **确认航站楼** | 有些机场有多个航站楼，确认两段航班是否在同一个 |
| **提前查政策** | 转机城市的签证/免签政策可能随时变化，出发前再确认一次 |
| **带好文件** | 护照、美国签证页、后续航班确认单（打印或手机截图） |

### 你的最佳方案

> 鉴于你只有美签，没有日本签证和港澳通行证：

| 推荐度 | 方案 | 原因 |
|:---:|------|------|
| ⭐⭐⭐⭐⭐ | **直飞**上海→LA | 最省心，不用考虑转机 |
| ⭐⭐⭐⭐⭐ | **首尔转机** | 不出机场不需要签证，机场体验好 |
| ⭐⭐⭐⭐ | **东京成田转机**（同机场） | 不出机场不需要签证，但确认同一机场 |
| ⭐⭐⭐ | **香港转机**（去程） | 不出机场不需要通行证，但回程要注意 |
| ⚠️ | **香港转机**（回程） | 可能需要港澳通行证，需确认 |
| ❌ | **东京换机场** | 需要日本签证，别选 |
